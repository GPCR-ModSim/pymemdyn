create_itp(ligand: str, charge: int) → None

Call ligpargen to create gromacs itp file and corresponding openmm pdb file. Note that original pdb file will be replaced by opnemm pdb file.

Parameters:

  • pdbfile – string containing local path to pdb of molecule. In commandline -i.

  • charge – interger charge of molecule. In commandline -c.

  • numberOfOptimizations – number of optimizations done by ligpargen. In cmdline -o.

Returns:

None

Writes itp file and new pdf file to current dir. old pdb is saved in dir ligpargenInput. unneccessary ligpargen output is saved in dir ligpargenOutput.
